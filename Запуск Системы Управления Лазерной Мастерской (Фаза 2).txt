## 🍓 ВАРИАНТ 1: Запуск на Raspberry Pi 3 B+

### 1.1 Подготовка системы

```bash
# Обновление системы
sudo apt update && sudo apt upgrade -y

# Установка Python 3 и зависимостей
sudo apt install -y python3 python3-pip python3-venv git

# Создание директории проекта
mkdir -p ~/laser-workshop-phase2
cd ~/laser-workshop-phase2

# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install flask pyjwt requests
```

### 1.2 Размещение файлов

```bash
# Структура проекта
laser-workshop-phase2/
├── app.py                    # Flask сервер (Фаза 2)
├── bot_worker.py             # VK бот (Фаза 2)
├── templates/
│   └── index.html           # Web-интерфейс (Фаза 2)
├── backups/                  # Папка для бэкапов (создаётся автоматически)
└── venv/                     # Виртуальное окружение
```

### 1.3 Настройка VK API

```bash
# Редактирование app.py
nano app.py

# Заменить строки:
VK_TOKEN = 'YOUR_VK_TOKEN_HERE'      # → Ваш токен VK
VK_GROUP_ID = 'YOUR_GROUP_ID'        # → ID вашей группы
```

**Получение токена VK:**
1. Зайдите на https://vk.com/dev
2. Создайте Standalone-приложение
3. Получите Access Token с правами: `messages`, `offline`
4. ID группы: https://vk.com/clubXXXXXXXXX (число после club)

### 1.4 Настройка автозапуска (systemd)

```bash
# Создание сервиса для Flask
sudo nano /etc/systemd/system/laser-workshop.service
```

```ini
[Unit]
Description=Laser Workshop CRM Phase 2
After=network.target

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/laser-workshop-phase2
Environment="PATH=/home/pi/laser-workshop-phase2/venv/bin"
ExecStart=/home/pi/laser-workshop-phase2/venv/bin/python3 app.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
# Создание сервиса для бота
sudo nano /etc/systemd/system/laser-bot.service
```

```ini
[Unit]
Description=Laser Workshop VK Bot Phase 2
After=network.target laser-workshop.service

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/laser-workshop-phase2
Environment="PATH=/home/pi/laser-workshop-phase2/venv/bin"
ExecStart=/home/pi/laser-workshop-phase2/venv/bin/python3 bot_worker.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
# Включение сервисов
sudo systemctl daemon-reload
sudo systemctl enable laser-workshop.service
sudo systemctl enable laser-bot.service
sudo systemctl start laser-workshop.service
sudo systemctl start laser-bot.service

# Проверка статуса
sudo systemctl status laser-workshop.service
sudo systemctl status laser-bot.service
```

### 1.5 Настройка сети

```bash
# Узнать IP адрес
hostname -I

# Настройка статического IP (опционально)
sudo nano /etc/dhcpcd.conf
```

```ini
interface eth0
static ip_address=192.168.1.100/24
static routers=192.168.1.1
static domain_name_servers=8.8.8.8 1.1.1.1
```

```bash
sudo systemctl restart dhcpcd
```

## ⚠️ РЕШЕНИЕ ПРОБЛЕМ

| Проблема | Решение |
|----------|---------|
| **Port 5000 занят** | Изменить порт в `app.run(port=5001)` |
| **VK бот не работает** | Проверить токен и права доступа |
| **БД не создаётся** | Проверить права на запись в папку |
| **Нет доступа из сети** | Проверить брандмауэр и IP |
| **Промокоды не работают** | Проверить таблицу `promo_codes` в БД |
| **Календарь пустой** | Добавить заказы с `planned_date` |

### Диагностика БД

```bash
# Raspberry Pi
sqlite3 workshop.db ".tables"
sqlite3 workshop.db "SELECT * FROM promo_codes;"

# Windows
.\venv\Scripts\python.exe
>>> import sqlite3
>>> conn = sqlite3.connect('workshop.db')
>>> conn.execute("SELECT * FROM promo_codes").fetchall()
```

### Восстановление БД

```bash
# Raspberry Pi
cd ~/laser-workshop-phase2
cp backups/workshop_backup_YYYYMMDD.db workshop.db
sudo systemctl restart laser-workshop.service

# Windows
cd C:\laser-workshop-phase2
copy backups\workshop_backup_YYYYMMDD.db workshop.db
# Перезапустить сервисы
```
