# 🔬 Лазерная Мастерская CRM (Фаза 3)

Система управления лазерной мастерской с полным циклом автоматизации.

## Функционал

### Основные возможности:
- **Управление заказами** - создание, редактирование, отслеживание статусов
- **Клиентская база** - сегментация клиентов, история заказов
- **Кэшбек система** - начисление и списание бонусов
- **Ролевая модель** - admin, manager, master
- **Прайс-лист** - гибкая система расчёта стоимости
- **Складской учёт** - мониторинг материалов
- **Аналитика** - дашборды, графики, отчёты
- **PWA** - прогрессивное веб-приложение
- **AI прогнозы** - предсказание загрузки и выручки
- **Интеграции**:
  - VKontakte (бот для приёма заказов)
  - ЮKassa (платежи)
  - CDEK (доставка)

## Установка

```bash
cd laser_workshop
pip install -r requirements.txt
python app.py
```

## Запуск

1. Основной сервер (веб-интерфейс):
```bash
python app.py
```
Доступ: http://localhost:5000
Логин: `admin`, Пароль: `admin123`

2. VK бот (опционально):
```bash
python bot_worker.py
```

## Конфигурация

Отредактируйте следующие переменные в `app.py`:
- `SECRET_KEY` - секретный ключ для JWT
- `VK_TOKEN` - токен VK API
- `VK_GROUP_ID` - ID группы VK
- `YOOKASSA_SECRET` - секрет ЮKassa
- `CDEK_API_KEY` - ключ CDEK API

## База данных

При первом запуске автоматически создаётся SQLite база данных `workshop.db`.

## Экспорт и резервное копирование

- Скачать резервную копию БД: `/api/backup/download`
- Экспорт заказов в CSV: `/api/export/csv`
