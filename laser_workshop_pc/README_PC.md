# 🖥️ Laser Workshop PC — Версия для Windows/Mac/Linux

Оптимизированная версия системы управления лазерной мастерской для персональных компьютеров.

## ⚡ Оптимизации для быстрого запуска

| Параметр | Значение | Эффект |
|----------|----------|--------|
| **Пул соединений** | 10 предсозданных | Мгновенный доступ к БД |
| **isolation_level** | None (autocommit) | +30% скорость записи |
| **PRAGMA busy_timeout** | 30000ms | Защита от блокировок |
| **PRAGMA journal_mode** | WAL | Быстрые транзакции |
| **PRAGMA cache_size** | 64MB | Кэширование в RAM |
| **THREAD_POOL** | 5 потоков | Параллельная обработка |
| **AI_PROGNOSIS** | ВКЛ | Прогнозирование загрузки |
| **DEBUG_MODE** | ВКЛ | Отладка в реальном времени |

## 📁 Структура проекта

```
laser_workshop_pc/
├── app.py              # Основное Flask-приложение
├── bot_worker.py       # VK бот (опционально)
├── requirements.txt    # Зависимости Python
├── start.bat           # Скрипт быстрого запуска (Windows)
└── README_PC.md        # Эта документация
```

## 🚀 Быстрый старт

### Вариант 1: Двойной клик (Windows)
1. Откройте папку `laser_workshop_pc`
2. Дважды кликните на `start.bat`
3. Готово! Сервер запущен

### Вариант 2: Через командную строку
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сервера
python app.py

# Запуск VK бота (в отдельном окне, опционально)
python bot_worker.py
```

## ⚙️ Настройка

### 1. Токен VK (для работы бота)
Откройте `app.py` и `bot_worker.py`, замените:
```python
VK_TOKEN = 'YOUR_VK_TOKEN_HERE'
VK_GROUP_ID = 'YOUR_GROUP_ID'
```

### 2. Изменение параметров оптимизации
В начале `app.py`:
```python
MAX_CONNECTIONS = 10      # Количество соединений в пуле
THREAD_POOL_SIZE = 5      # Размер пула потоков
CACHE_SIZE = 64*1024*1024 # Кэш БД (64MB)
AI_PROGNOSIS = True       # Включить AI прогнозы
DEBUG_MODE = True         # Режим отладки
```

## 🛠️ Автозапуск при старте Windows

### Способ 1: Планировщик заданий
1. Откройте "Планировщик заданий"
2. Создайте задачу с триггером "При входе в систему"
3. Действие: `C:\Python39\python.exe C:\path\to\app.py`

### Способ 2: Папка автозагрузки
1. Создайте ярлык для `start.bat`
2. Поместите в `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`

## 🔧 Troubleshooting

### Ошибка "Port 5000 already in use"
```bash
# Найдите процесс на порту 5000
netstat -ano | findstr :5000

# Убейте процесс (замените PID)
taskkill /PID 1234 /F
```

### Ошибка "Module not found"
```bash
pip install --upgrade -r requirements.txt
```

### Бот VK не работает
1. Проверьте токен в VK App (https://vk.com/dev)
2. Убедитесь, что у токена есть права на сообщения
3. Проверьте ID группы (должен быть числом)
