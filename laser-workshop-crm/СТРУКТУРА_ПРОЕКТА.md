## 🔧 Установка и запуск

### 1. Создание виртуального окружения
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate     # Windows
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
```bash
cp .env.example .env
# Отредактируйте .env, указав ваши токены и настройки
```

### 4. Запуск
```bash
# Запуск админ панели
python app.py

# Запуск бота (в отдельном терминале)
python bot_worker.py

# Или используйте скрипты
./start.sh      # Linux
start.bat       # Windows
```

## 📦 Основные компоненты

### VK Бот (`laser_workshop/bot/`)
- Обработка сообщений от клиентов и администраторов
- Машина состояний для многошаговых сценариев
- Генерация клавиатур (inline и button)
- Интеграция с базой данных

### Админ Панель (`laser_workshop/admin/`)
- Flask веб-интерфейс
- Управление заказами, клиентами, услугами
- Складской учёт
- Отчёты и аналитика
- REST API для интеграций

### Ядро (`laser_workshop/core/`)
- Централизованная конфигурация
- Работа с базой данных
- Общие утилиты и хелперы

### Сервисы (`laser_workshop/services/`)
- Уведомления (VK, email, SMS)
- Платежи (ЮKassa, CDEK)
- Аналитика и отчёты
- Резервное копирование

## 🔐 Безопасность

- JWT токены для API
- Хеширование паролей (bcrypt)
- Проверка прав доступа (роли: admin, manager, client)
- HTTPS поддержка
- Rate limiting для API
