## 🍓 ЧАСТЬ 1: Raspberry Pi 3 Model B+

### 1.1 Подготовка Raspberry Pi OS

```bash
# Обновление системы
sudo apt update && sudo apt upgrade -y

# Установка Python 3 и зависимостей
sudo apt install -y python3 python3-pip python3-venv git

# Создание директории проекта
mkdir -p ~/laser-workshop
cd ~/laser-workshop

# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install flask pyjwt requests
```

### 1.2 Оптимизация для Raspberry Pi 3 B+

```bash
# Создание файла оптимизации
cat > ~/laser-workshop/optimize.sh << 'EOF'
#!/bin/bash
# Отключение лишних сервисов для экономии RAM
sudo systemctl disable bluetooth
sudo systemctl disable wifi
sudo systemctl disable triggerhappy
sudo systemctl disable dphys-swapfile

# Увеличение swap файла (для 1GB RAM)
sudo dphys-swapfile swapoff
sudo sed -i 's/CONF_SWAPSIZE=100/CONF_SWAPSIZE=512/' /etc/dphys-swapfile
sudo dphys-swapfile setup
sudo dphys-swapfile swapon

# Установка лимитов для Flask
echo "export FLASK_ENV=production" >> ~/.bashrc
echo "export WORKERS=2" >> ~/.bashrc
EOF

chmod +x ~/laser-workshop/optimize.sh
sudo ~/laser-workshop/optimize.sh
```

### 1.3 Настройка автозапуска

```bash
# Создание systemd сервиса
sudo nano /etc/systemd/system/laser-workshop.service
```

```ini
[Unit]
Description=Laser Workshop CRM
After=network.target

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/laser-workshop
Environment="PATH=/home/pi/laser-workshop/venv/bin"
ExecStart=/home/pi/laser-workshop/venv/bin/python3 app.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
# Создание сервиса для бота
sudo nano /etc/systemd/system/laser-bot.service
```

```ini
[Unit]
Description=Laser Workshop VK Bot
After=network.target laser-workshop.service

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/laser-workshop
Environment="PATH=/home/pi/laser-workshop/venv/bin"
ExecStart=/home/pi/laser-workshop/venv/bin/python3 bot_worker.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
# Включение сервисов
sudo systemctl daemon-reload
sudo systemctl enable laser-workshop.service
sudo systemctl enable laser-bot.service
sudo systemctl start laser-workshop.service
sudo systemctl start laser-bot.service

# Проверка статуса
sudo systemctl status laser-workshop.service
sudo systemctl status laser-bot.service
```

### 1.4 Настройка сети и доступа

```bash
# Узнать IP адрес Raspberry Pi
hostname -I

# Настройка статического IP (опционально)
sudo nano /etc/dhcpcd.conf
```

```ini
interface eth0
static ip_address=192.168.1.100/24
static routers=192.168.1.1
static domain_name_servers=8.8.8.8 1.1.1.1
```

```bash
# Перезапуск сети
sudo systemctl restart dhcpcd
```

### 1.5 Мониторинг ресурсов

```bash
# Скрипт мониторинга
cat > ~/laser-workshop/monitor.sh << 'EOF'
```

## ⚙️ ЧАСТЬ 3: Конфигурация

### 3.1 Настройка VK API

```python
# В app.py и bot_worker.py заменить:
VK_TOKEN = 'YOUR_VK_TOKEN_HERE'  # → Ваш токен VK
VK_GROUP_ID = 'YOUR_GROUP_ID'    # → ID вашей группы
```

**Получение токена VK:**
1. Зайдите на https://vk.com/dev
2. Создайте Standalone-приложение
3. Получите Access Token с правами: `messages`, `offline`
4. ID группы: https://vk.com/clubXXXXXXXXX (число после club)

### 3.2 Структура проекта

```

├── app.py                 # Flask сервер
```

## ⚠️ ЧАСТЬ 7: Решение проблем

### 7.1 Частые ошибки

| Проблема | Решение |
|----------|---------|
| **Port 5000 уже занят** | Изменить порт в `app.run(port=5001)` |
| **VK бот не работает** | Проверить токен и права доступа |
| **Raspberry Pi перегрев** | Добавить радиатор/вентилятор |
| **Нехватка RAM на Pi** | Увеличить swap, отключить лишние сервисы |
| **БД повреждена** | Восстановить из backups/ |
| **Нет доступа из сети** | Проверить брандмауэр и IP |

### 7.2 Диагностика

```bash
# Raspberry Pi - проверка
python3 -c "import flask; print(flask.__version__)"
sudo systemctl status laser-workshop.service
netstat -tlnp | grep 5000

# Windows - проверка
python -c "import flask; print(flask.__version__)"
netstat -an | findstr 5000
```

### 7.3 Восстановление БД

```bash
# Raspberry Pi
cd ~/laser-workshop
cp backups/workshop_backup_YYYYMMDD.db workshop.db
sudo systemctl restart laser-workshop.service

# Windows
cd C:\laser-workshop
copy backups\workshop_backup_YYYYMMDD.db workshop.db
# Перезапустить сервисы
```
