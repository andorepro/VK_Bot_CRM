## 🍓 ВАРИАНТ 1: Запуск на Raspberry Pi 3 B+

### 1.1 Подготовка системы

```bash
# Обновление системы
sudo apt update && sudo apt upgrade -y

# Установка Python 3 и зависимостей
sudo apt install -y python3 python3-pip python3-venv git sqlite3

# Создание директории проекта
mkdir -p ~/laser-workshop-phase3
cd ~/laser-workshop-phase3

# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install flask pyjwt requests
```

### 1.2 Размещение файлов

```bash
# Структура проекта
laser-workshop-phase3/
├── app.py                    # Flask сервер (Фаза 3)
├── bot_worker.py             # VK бот (Фаза 3)
├── templates/
│   └── index.html           # Web-интерфейс (Фаза 3)
├── static/                   # Статические файлы (PWA)
│   ├── icon-192.png
│   └── icon-512.png
├── backups/                  # Папка для бэкапов
└── venv/                     # Виртуальное окружение
```

```bash
# Копирование файлов
cp app(фаза 3).py app.py
cp bot_worker(фаза 3).py bot_worker.py
mkdir -p templates static backups
cp index(фаза 3).html templates/index.html
```

### 1.3 Настройка VK API

```bash
# Редактирование app.py и bot_worker.py
nano app.py
nano bot_worker.py

# Заменить строки в обоих файлах:
VK_TOKEN = 'YOUR_VK_TOKEN_HERE'      # → Ваш токен VK
VK_GROUP_ID = 'YOUR_GROUP_ID'        # → ID вашей группы
```

**Получение токена VK:**
1. Зайдите на https://vk.com/dev
2. Создайте Standalone-приложение
3. Получите Access Token с правами: `messages`, `offline`
4. ID группы: https://vk.com/clubXXXXXXXXX (число после club)

### 1.4 Настройка автозапуска (systemd)

```bash
# Создание сервиса для Flask
sudo nano /etc/systemd/system/laser-workshop.service
```

```ini
[Unit]
Description=Laser Workshop CRM Phase 3
After=network.target

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/laser-workshop-phase3
Environment="PATH=/home/pi/laser-workshop-phase3/venv/bin"
ExecStart=/home/pi/laser-workshop-phase3/venv/bin/python3 app.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
# Создание сервиса для бота
sudo nano /etc/systemd/system/laser-bot.service
```

```ini
[Unit]
Description=Laser Workshop VK Bot Phase 3
After=network.target laser-workshop.service

[Service]
Type=simple
User=pi
WorkingDirectory=/home/pi/laser-workshop-phase3
Environment="PATH=/home/pi/laser-workshop-phase3/venv/bin"
ExecStart=/home/pi/laser-workshop-phase3/venv/bin/python3 bot_worker.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
# Включение сервисов
sudo systemctl daemon-reload
sudo systemctl enable laser-workshop.service
sudo systemctl enable laser-bot.service
sudo systemctl start laser-workshop.service
sudo systemctl start laser-bot.service

# Проверка статуса
sudo systemctl status laser-workshop.service
sudo systemctl status laser-bot.service
```

### 1.5 Оптимизация для Raspberry Pi

```bash
# Увеличение swap файла (для 1GB RAM)
sudo dphys-swapfile swapoff
sudo sed -i 's/CONF_SWAPSIZE=100/CONF_SWAPSIZE=1024/' /etc/dphys-swapfile
sudo dphys-swapfile setup
sudo dphys-swapfile swapon

# Отключение лишних сервисов
sudo systemctl disable bluetooth
sudo systemctl disable triggerhappy
```

### 1.6 Настройка сети

```bash
# Узнать IP адрес
hostname -I

# Настройка статического IP (опционально)
sudo nano /etc/dhcpcd.conf
```

```ini
interface eth0
static ip_address=192.168.1.100/24
static routers=192.168.1.1
static domain_name_servers=8.8.8.8 1.1.1.1
```

```bash
sudo systemctl restart dhcpcd
```

## 📱 PWA (Progressive Web App)

### Установка на устройство

1. Откройте `http://[IP]:5000` в браузере
2. Нажмите **"Установить приложение"** (появится автоматически)
3. Приложение появится на рабочем столе

### Офлайн режим

- При потере соединения появится индикатор
- Данные кэшируются локально
- Синхронизация при восстановлении соединения

### Файлы для PWA

```
static/
├── icon-192.png    # Иконка 192x192
├── icon-512.png    # Иконка 512x512
manifest.json       # Манифест приложения
sw.js              # Service Worker
```

## ⚠️ РЕШЕНИЕ ПРОБЛЕМ

| Проблема | Решение |
|----------|---------|
| **Port 5000 занят** | Изменить порт в `app.run(port=5001)` |
| **VK бот не работает** | Проверить токен и права доступа |
| **БД не создаётся** | Проверить права на запись в папку |
| **Нет доступа из сети** | Проверить брандмауэр и IP |
| **PWA не устанавливается** | Проверить HTTPS (требуется для PWA) |
| **AI прогнозы пустые** | Нужно минимум 7 дней истории |
| **Склад не списывается** | Проверить названия услуг в БД |
| **Кэшбек не начисляется** | Статус заказа должен быть DONE |

### Диагностика БД

```bash

# Windows

>>> conn = sqlite3.connect('workshop.db')
>>> conn.execute("SELECT * FROM inventory").fetchall()
```

## 📞 ПОДДЕРЖКА

При возникновении проблем:

1. **Проверьте логи сервисов**
   ```bash
   sudo journalctl -u laser-workshop.service -n 100
   ```

2. **Убедитесь что VK токен действителен**
   - Проверьте права доступа
   - Проверьте срок действия токена

3. **Проверьте права доступа к папкам**
   ```bash
   ls -la ~/laser-workshop-phase3/
   chmod -R 755 ~/laser-workshop-phase3/
   ```

4. **Восстановите БД из бэкапа при необходимости**
