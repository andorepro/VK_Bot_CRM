## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
cd laser-workshop-crm
pip install -r requirements.txt
```

### 2. Настройка окружения
Создайте файл `.env`:
```bash
cp .env.example .env
```

Отредактируйте `.env`:
```
VK_TOKEN=ваш_token_группы_ВК
ADMIN_IDS=12345678,87654321
SECRET_KEY=ваш_секретный_ключ
DEFAULT_ADMIN_PASSWORD=admin123
DB_PATH=workshop.db
```

### 3. Создание администратора
При первом запуске создаётся пользователь по умолчанию:
- Логин: `admin`
- Пароль: значение из `DEFAULT_ADMIN_PASSWORD`

### 4. Запуск сервера
```bash
# Разработка
python app.py

# Production (с gunicorn)
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```

### 5. Запуск бота
```bash
python bot_worker.py
```

### 6. Доступ к CRM
Откройте в браузере: `http://localhost:5000`

## 🛠️ API Endpoints

### Аутентификация
- `GET /login` — страница входа
- `POST /login` — аутентификация
- `GET /logout` — выход

### Заказы
- `GET /api/orders` — список заказов
- `POST /api/orders` — создание заказа
- `PUT /api/orders/<id>` — обновление заказа
- `DELETE /api/orders/<id>` — удаление заказа

### VK Integration
- `GET /api/vk/longpoll/init` — инициализация Long Poll
- `POST /api/chat/receive` — получение сообщений
- `POST /api/chat/send` — отправка сообщений
- `GET /api/chat/history` — история чата

### Клиенты
- `GET /api/clients` — список клиентов
- `GET /api/clients/<id>` — информация о клиенте
- `GET /api/clients/<id>/cashback` — кэшбек клиента

### Склад
- `GET /api/stock` — список материалов
- `POST /api/stock` — добавление материала
- `PUT /api/stock/<id>` — обновление количества

## 📊 База данных

### Таблицы:
1. `users` — пользователи системы
2. `clients` — клиенты с кэшбеком
3. `orders` — заказы
4. `order_items` — позиции заказов
5. `stock` — склад материалов
6. `cashback_transactions` — операции кэшбека
7. `settings` — настройки системы
8. `audit_log` — журнал аудита
9. `price_list` — прайс-лист
10. `discounts` — скидки и промокоды
11. `notifications` — уведомления
12. `analytics_cache` — кеш аналитики
