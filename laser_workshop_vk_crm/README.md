# 🚀 Laser Workshop VK CRM

Полностью готовая система управления лазерной мастерской с интеграцией ВКонтакте.

## 📋 Возможности

- **11 алгоритмов расчета** стоимости (фиксированная, площадь, резка, гравировка, B2B и др.)
- **VK Бот** для автоматического приема заказов через сообщения сообщества
- **Web-CRM панель** с дашбордом, управлением заказами и live-чатом
- **Адаптивный дизайн** для ПК и мобильных устройств
- **HTTPS поддержка** с самоподписанными сертификатами
- **Автоматические уведомления** клиентам при смене статуса заказа
- **Бэкапы базы данных** в один клик

## 🏗️ Архитектура

## ⚡ Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Откройте файл `.env` и укажите ваши данные:

```ini
SECRET_KEY=ваш_секретный_ключ
VK_TOKEN=токен_сообщества_VK
VK_GROUP_ID=id_группы_VK
```

**Как получить VK Token:**
1. Зайдите в [VK Developers](https://vk.com/dev)
2. Создайте приложение типа "Community"
3. Получите токен с правами `messages`, `groups`

### 3. Генерация SSL сертификатов

```bash
# Linux/Mac
./generate_certs.sh

# Windows (через Git Bash или WSL)
bash generate_certs.sh
```

### 4. Запуск системы

```bash
# Linux/Raspberry Pi
./start.sh

# Windows
start.bat
```

## 🤖 Как работает бот

1. Клиент пишет сообществу ВКонтакте
2. Бот показывает меню услуг (11 вариантов)
3. Клиент выбирает услугу и вводит параметры
4. Бот рассчитывает стоимость и создает заказ
5. При статусе "DONE" клиент получает уведомление

## 📊 Типы расчетов

1. **fixed** - Штучный товар (фляжки, жетоны)
2. **area_cm2** - Шильды/дерево (площадь × цена)
3. **meter_thickness** - Резка фанеры (длина × толщина)
4. **per_minute** - Гравировка MOPA (время × цена)
5. **per_char** - Кольца/ручки (символы × цена)
6. **vector_length** - Векторная резка (метры × цена)
7. **setup_batch** - B2B тираж (настройка + шт × цена со скидкой)
8. **photo_raster** - Фото (площадь × DPI множитель)
9. **cylindrical** - Термосы (диаметр × длина)
10. **volume_3d** - 3D клише (объем × цена)
11. **material_and_cut** - Материал + резка (комплекс)

## 📝 Структура БД

**orders** - заказы:
- `description` - плоский текст с параметрами (без JSON)
- `status` - NEW → PROCESSING → DONE → DELIVERED

**clients** - клиенты из VK:
- `vk_id`, `first_name`, `last_name`
- `total_orders`, `total_spent` - статистика

**price_list** - услуги:
- 11 записей с типами расчетов

## 🔧 Troubleshooting

**Ошибка SSL:**
```bash
# Перегенерируйте сертификаты
rm -rf certs/*
./generate_certs.sh
```

**Бот не работает:**
- Проверьте `VK_TOKEN` и `VK_GROUP_ID` в `.env`
- Убедитесь, что у токена есть права `messages`, `groups`
- Проверьте подключение к интернету

**Сервер не запускается:**
```bash
# Проверьте логи
python app.py

# Убедитесь, что порт свободен
netstat -an | grep 5000
```
