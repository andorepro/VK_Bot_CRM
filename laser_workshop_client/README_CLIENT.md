# 🔬 Лазерная Мастерская CRM - Клиент для ПК и мобильных

## 📋 Описание

Клиентское приложение для работы с сервером Laser Workshop на Raspberry Pi. Оптимизировано для:
- **ПК (Windows/Mac/Linux)** - полноэкранный режим, горячие клавиши
- **Мобильных устройств (iOS/Android)** - PWA, офлайн-режим, сенсорное управление
- **Планшетов** - адаптивный интерфейс

## ⚡ Быстрый старт

### 1. Настройка конфигурации

```bash
cd laser_workshop_client
cp .env.example .env
```

Отредактируйте `.env`:

```ini
# IP-адрес вашего Raspberry Pi в локальной сети
SERVER_HOST=192.168.1.100
SERVER_PORT=5000

# Порт клиента (отличный от сервера)
PORT=5001

# Режим работы
OFFLINE_MODE=False
AUTO_SYNC=True
SYNC_INTERVAL_SECONDS=30
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Запуск клиента

```bash
python app.py
```

### 4. Открыть в браузере

- **ПК:** http://localhost:5001
- **С телефона в той же сети:** http://<IP_ПК>:5001

## 🔧 Конфигурация

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `SERVER_HOST` | 192.168.1.100 | IP адрес Raspberry Pi |
| `SERVER_PORT` | 5000 | Порт сервера на RPi |
| `PORT` | 5001 | Порт клиента |
| `OFFLINE_MODE` | False | Работа без подключения к серверу |
| `AUTO_SYNC` | True | Автоматическая синхронизация |
| `SYNC_INTERVAL_SECONDS` | 30 | Интервал синхронизации |
| `CACHE_TTL_SECONDS` | 60 | Время жизни кэша |
| `THEME` | dark | Тема интерфейса (dark/light) |
| `MOBILE_OPTIMIZED` | True | Мобильная оптимизация |

## 📱 Возможности клиента

### Для ПК:
- ✅ Полноэкранный режим
- ✅ Горячие клавиши (Ctrl+S - сохранить, Ctrl+N - новый заказ)
- ✅ Быстрый поиск по заказам и клиентам
- ✅ Экспорт данных в CSV/Excel
- ✅ Печать этикеток и документов

### Для мобильных:
- ✅ PWA (установка на главный экран)
- ✅ Офлайн-режим с последующей синхронизацией
- ✅ Сенсорное управление
- ✅ Push-уведомления (при поддержке браузера)
- ✅ Камера для сканирования QR-кодов заказов

### Общие:
- ✅ Темная/светлая тема
- ✅ Адаптивный дизайн
- ✅ Мгновенная загрузка из кэша
- ✅ Индикатор соединения с сервером
- ✅ Автоматическое переподключение

## 🔄 Синхронизация с сервером

### Автоматическая синхронизация:
1. Каждые 30 секунд проверяется доступность сервера
2. Офлайн-операции отправляются на сервер при подключении
3. Кэш обновляется при изменении данных

### Ручная синхронизация:
```javascript
// В консоли браузера
fetch('/api/sync', {method: 'POST'})
```

## 🐛 Решение проблем

### "Server unavailable"
1. Проверьте, что Raspberry Pi включен
2. Убедитесь, что IP адрес верный
3. Проверьте команду: `ping 192.168.1.100`

### Кэш не обновляется
```bash
# Очистить кэш вручную
rm client_cache.db
# Перезапустить клиент
python app.py
```

### PWA не устанавливается
1. Откройте сайт в Chrome/Safari
2. Нажмите "Поделиться" → "На экран «Домой»"
3. Или в меню браузера: "Установить приложение"
