# Система Управления Лазерной Мастерской

Веб-приложение для управления заказами, клиентами и производством в лазерной мастерской.

## Описание

Это многофункциональная система для автоматизации работы лазерной мастерской, включающая:
- Управление заказами и клиентами
- Расчёт стоимости работ
- Интеграция с ВКонтакте (бот)
- Генерация отчётов и документов
- Резервное копирование данных

## Требования

- Python 3.8+
- Flask
- PyJWT
- Requests
- SQLite3 (встроен в Python)

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd <project-directory>
```

2. Установите зависимости:
```bash
pip install flask pyjwt requests
```

3. Настройте конфигурацию в `app.py`:
```python
SECRET_KEY = 'your_secret_key_here'
VK_TOKEN = 'your_vk_token_here'
VK_GROUP_ID = 'your_group_id_here'
```

## Запуск

### Фаза 1 (Базовая версия)
```bash
python app.py
```

### Фаза 2 (Расширенная версия)
```bash
python "app(фаза 2).py"
```

### Фаза 3 (Полная версия с интеграцией)
```bash
python "app(фаза 3).py"
```

После запуска откройте браузер и перейдите по адресу: `http://localhost:5000`

## Запуск бота ВКонтакте

```bash
python bot_worker.py
# или для других фаз:
python "bot_worker(фаза 2).py"
python "bot_worker(фаза 3).py"
```

## Функционал

### Основные возможности:
- **Аутентификация** - система входа для администраторов
- **Управление заказами** - создание, редактирование, удаление заказов
- **Клиентская база** - хранение информации о клиентах
- **Расчёт стоимости** - автоматический расчёт на основе параметров
- **Отчётность** - экспорт данных в CSV/XLSX
- **Резервное копирование** - автоматическое создание бэкапов БД
- **Интеграция с ВК** - приём заказов через сообщения группы

### Этапы разработки:
- **Фаза 1** - Базовый функционал управления заказами
- **Фаза 2** - Расширенные функции и улучшенный интерфейс
- **Фаза 3** - Полная интеграция с социальными сетями и дополнительные модули

## База данных

Приложение использует SQLite. Файл базы данных `workshop.db` создаётся автоматически при первом запуске.

Папка `backups/` содержит автоматические резервные копии базы данных.
