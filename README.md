# 🏭 Система Управления Лазерной Мастерской (Laser Workshop CRM)

Комплексная система автоматизации для лазерной мастерской с полным циклом управления заказами, клиентами и производством.

## 📋 О Проекте

Система предоставляет веб-интерфейс для управления мастерской, интеграцию с VKontakte для приёма заказов, платёжными системами и службами доставки.

### Основные возможности

- **Управление заказами** — создание, редактирование, отслеживание статусов
- **Клиентская база** — сегментация клиентов, история заказов
- **Кэшбек система** — начисление и списание бонусов
- **Ролевая модель** — admin, manager, master
- **Прайс-лист** — гибкая система расчёта стоимости
- **Складской учёт** — мониторинг материалов
- **Аналитика** — дашборды, графики, отчёты
- **PWA** — прогрессивное веб-приложение для мобильных устройств
- **AI прогнозы** — предсказание загрузки и выручки

### Интеграции

- **VKontakte** — бот для приёма заказов
- **ЮKassa** — приём платежей
- **CDEK** — расчёт и оформление доставки

## 🚀 Быстрый Старт

### Вариант 1: Основная версия (laser_workshop)

```bash
cd laser_workshop
pip install -r requirements.txt
python app.py
```

Доступ: http://localhost:5000  
Логин: `admin`, Пароль: `admin123`

### Вариант 2: Клиент-серверная архитектура

**Сервер (Raspberry Pi):**

## ⚙️ Конфигурация

### Переменные окружения

Основные параметры настраиваются через переменные окружения или файлы `.env`:

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| `SECRET_KEY` | Секретный ключ для JWT | (требуется установка) |
| `VK_TOKEN` | Токен VK API | (опционально) |
| `VK_GROUP_ID` | ID группы VK | (опционально) |
| `YOOKASSA_SECRET` | Секрет ЮKassa | (опционально) |
| `CDEK_API_KEY` | Ключ CDEK API | (опционально) |
| `PORT` | Порт сервера | 5000 |
| `HOST` | Хост сервера | 0.0.0.0 |

### Генерация SSL сертификатов

Для HTTPS соединения выполните:

```bash
./generate_certs.sh <путь_к_папке_сертификатов>
```

## 📱 PWA (Progressive Web App)

Система поддерживает установку как PWA:

1. Откройте сайт в браузере (Chrome/Safari)
2. Нажмите «Добавить на главный экран»
3. Используйте как нативное приложение

### Доступ с мобильного устройства

1. Узнайте IP компьютера: `ipconfig` (Windows) или `ifconfig` (Linux/Mac)
2. Откройте в браузере телефона: `https://<IP>:5001`
3. Примите сертификат безопасности
4. Добавьте на главный экран
